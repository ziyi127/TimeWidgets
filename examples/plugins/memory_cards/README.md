# 记忆卡片插件

这是一个自动轮播单词等记忆内容的卡片插件，适用于TimeWidgets应用。

## 功能特性

- 自动轮播单词和释义
- 支持自定义单词库
- 美观的卡片式界面
- 可调整轮播间隔

## 如何使用

1. 将此文件夹压缩为ZIP文件
2. 在TimeWidgets应用的插件管理页面中，点击"安装插件"按钮
3. 选择压缩后的ZIP文件进行安装
4. 安装完成后，插件会自动出现在应用中

## 自定义单词库

1. 打开`data.json`文件
2. 在`cards`数组中添加或修改单词和释义
3. 保存文件后重新压缩并安装

## 配置选项

### 插件设置

在插件管理页面中，你可以通过设置页面调整以下选项：

- `自动播放`: 是否自动轮播记忆卡片
- `轮播间隔(秒)`: 卡片切换的时间间隔
- `卡片样式`: 选择卡片的外观风格（light/dark/colorful）
- `字体大小`: 调整单词的显示大小
- `显示释义`: 是否显示单词的释义
- `轮播内容`: 自定义要轮播的内容，每行一个条目
- `轮播描述`: 为每个轮播内容添加对应的描述
- `自动创建桌面小组件`: 启用时自动为插件创建桌面小组件

### 自定义单词库

在`data.json`文件中，你可以修改以下设置：

- `cards`: 单词和释义的数组
- `settings`: 插件的默认设置
  - `autoPlay`: 是否自动轮播（true/false）
  - `interval`: 轮播间隔时间（秒）

## 示例

```json
{
  "cards": [
    {
      "word": "Hello",
      "meaning": "你好"
    },
    {
      "word": "World",
      "meaning": "世界"
    }
  ],
  "settings": {
    "autoPlay": true,
    "interval": 5
  }
}
```

## 注意事项

- 请确保ZIP文件的根目录包含`plugin.json`文件
- 单词库不宜过大，建议不超过50个单词
- 轮播间隔建议设置在3-10秒之间，以保证良好的学习效果
